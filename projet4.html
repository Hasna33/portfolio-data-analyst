🌟 Projet 4 : A/B Testing & Statistiques sur les données d’Olist

🧩 Contexte et objectifs

Explorer un sous-ensemble du dataset Olist pour formuler et tester des hypothèses business avec des méthodes statistiques en Python.

📊 Données utilisées

  • Données Olist réduites : commandes, clients, produits, paiements, évaluations
  • Analyse dans un DataFrame pandas

🔍 Méthodologie

  • Exploration, nettoyage (types, valeurs manquantes, doublons)
  • Descriptives : stats centrales, distribution, délais, satisfaction
  • Corrélations : prix, délai, catégorie, satisfaction
  • A/B Testing : t-test, chi², ANOVA

📈 Résultats

  • Livraisons tardives = moins de satisfaction (p < 0.05)
  • Catégories influencent la note moyenne
  • Pas de différence significative entre méthodes de paiement

📌 Outils

  • Python : pandas, numpy, plotly, scipy
