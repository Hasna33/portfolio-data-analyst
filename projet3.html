🌟 Projet 3 : Analyse des données e-commerce avec Power BI (The Look Dataset)

🧩 Contexte et objectifs

Analyse d’un dataset complet simulant les opérations d’un site e-commerce de prêt-à-porter "The Look" pour identifier des leviers d’action stratégiques via Power BI.

📊 Données utilisées

  • Base multi-tables : commandes, produits, utilisateurs, inventaire, événements web, marketing…
  • Source : BigQuery → Power BI Desktop
  • Volume : plusieurs dizaines de milliers de lignes

🛠️ Compétences techniques mobilisées

  • Connexion BigQuery dans Power BI
  • Power Query Editor, modélisation relationnelle
  • Calcul de KPIs : revenus, taux de retour, conversion, ROI marketing
  • Segmentation, visualisations interactives

🧮 Démarche analytique

  • Étude du modèle e-commerce, sélection des axes clés : ventes, client, produit
  • Analyse des ventes, segmentation client, turnover produit
  • Création de dashboards dynamiques avec recommandations

📊 Résultats / Insights clés

  • 3 catégories = 65 % CA annuel
  • Retours concentrés sur certaines marques → problème qualité
  • Clients fidèles (3+ commandes) : panier moyen 40 % supérieur
  • Emailing : ROI 2x supérieur aux réseaux sociaux

💡 Recommandations

  • Optimiser les stocks sur les catégories à forte rotation
  • Réduire les canaux peu rentables, investir dans l’emailing
  • Renforcer les actions marketing ciblées sur les clients fidèles

🎓 Ce que j’ai appris

  • Manipuler un outil BI professionnel sur des données complexes
  • Créer une modélisation relationnelle dans Power BI
  • Faire émerger des recommandations à forte valeur business
